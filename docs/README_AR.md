# شنو هو المشروع؟

نظام زراعة ذكي يقرأ حالة النباتات والبيئة عن طريق حساسات، ويتخذ قرارات تلقائية (ري، تسميد، تهوية، تصحيح حموضة، إضاءة) بدون تدخل يدوي.

**الفكرة بجملة وحدة:**
> حساسات تقرأ → خوارزمية تفكر → أجهزة تنفذ تلقائياً

---

# كيف يشتغل؟

```
كل 5 ثواني:
  ESP32 يقرأ الحساسات → يرسل البيانات عبر WiFi
  
كل 30 ثانية:
  الخوارزمية تحلل البيانات وتقرر:
    - هل النبات يحتاج مياه؟ → شغل المضخة
    - هل الحرارة عالية؟ → شغل المروحة  
    - هل pH غلط؟ → شغل مضخة التصحيح
    - هل الضوء ناقص؟ → شغل الإضاءة
```

# ⚠️ هل لازم كل القطع موجودة؟

## ✅ لا! النظام يشتغل حتى بدون أي قطعة هاردوير!

### وضع المحاكاة (Simulation Mode) — بدون أي هاردوير

النظام فيه **وضع محاكاة كامل** يشتغل على أي لابتوب بدون ما تشتري أي قطعة:

```bash
# بس شغل المشروع وكل شي يشتغل بقيم وهمية
./scripts/run_all.sh
```

هذا الوضع:
- ✅ يولد بيانات حساسات واقعية وهمية (حرارة تتغير بالنهار والليل، تربة تنشف تدريجياً، الخ)
- ✅ الخوارزمية تشتغل وتتخذ قرارات حقيقية
- ✅ الداشبورد يعرض كل شي بالوقت الحقيقي
- ✅ تقدر تتعلم وتفهم النظام قبل ما تشتري أي شي

### الحساسات الاختيارية — شنو يصير لو ما جبت وحدة؟

| القطعة | لو ما جبتها | التأثير |
|--------|------------|---------|
| **DHT22** (حرارة/رطوبة) | ❌ مهمة جداً | النظام ما يقدر يتحكم بالمروحة والرذاذ |
| **حساس رطوبة التربة** | ❌ مهمة جداً | النظام ما يقدر يسقي تلقائياً |
| **حساس pH** | ⚠️ اختياري | بس ما يقدر يصحح الحموضة — تسويها يدوياً |
| **BH1750** (ضوء) | ⚠️ اختياري | ما يتحكم بالإضاءة التكميلية |
| **MH-Z19B** (CO₂) | ⚠️ اختياري | بس يخسر مراقبة ثاني أوكسيد الكربون |
| **DS18B20** (حرارة تربة) | ✅ اختياري | يستخدم حرارة الجو كبديل تقريبي |
| **INA219** (طاقة) | ✅ اختياري | بس يخسر مراقبة استهلاك الكهرباء |
| **Anemometer** (رياح) | ✅ اختياري | بس يخسر حماية الرذاذ من الرياح |
| **Rain Gauge** (مطر) | ✅ اختياري | بس يخسر كشف المطر لإيقاف الري |
| **مضخة الري** | ❌ أساسية | بدونها ما فيه ري تلقائي |
| **ريليات** | ❌ أساسية | بدونها ما يقدر يشغل أي جهاز |
| **مروحة** | ⚠️ اختياري | ما يتحكم بالحرارة |
| **إضاءة** | ✅ اختياري | للزراعة الداخلية فقط |
| **مضخة pH** | ✅ اختياري | تصحح الحموضة يدوياً |
| **مضخة سماد** | ✅ اختياري | تسمد يدوياً |

# التنصيب خطوة بخطوة

## الخطوة 1: تحضير الكمبيوتر (أو Raspberry Pi)

```bash
# حمّل المشروع
git clone https://github.com/sajjadraad2006-sys/Smart-Garden.git
cd Smart-Garden

# نصّب كل شي بأمر واحد
./scripts/install.sh
```

## الخطوة 2: التجربة بوضع المحاكاة

```bash
# شغل بدون هاردوير
./scripts/run_all.sh

# افتح الداشبورد
xdg-open agrimaster-pro.html

# افتح API Docs
xdg-open http://localhost:8000/docs
```

## الخطوة 3: برمجة ESP32 (لمن تجيب القطع)

1. نصب [Arduino IDE](https://www.arduino.cc/en/software)
2. عدل `firmware/config.h`:
```c
#define WIFI_SSID    "اسم_الواي_فاي"
#define WIFI_PASS    "باسورد_الواي_فاي"
#define MQTT_SERVER  "192.168.1.100"  // IP الكمبيوتر
```
3. ارفع الكود للـ ESP32 عبر USB

## الخطوة 4: توصيل الأسلاك

### الحد الأدنى (ري فقط):
```
ESP32          →    القطعة
─────────────────────────────
GPIO 36        →    DHT22 (بيانات) + مقاومة 10kΩ للـ 3.3V
GPIO 34        →    حساس رطوبة التربة (Analog)
GPIO 32        →    ريلي المضخة (IN)
GND            →    GND مشترك
3.3V           →    VCC الحساسات
5V             →    VCC الريلي
```

### التوصيل الكامل:
ارجع لملف [WIRING_DIAGRAM.md](WIRING_DIAGRAM.md)

## الخطوة 5: التشغيل الحقيقي

```bash
# غيّر من محاكاة لحقيقي
nano config/system.yaml
# غيّر: simulation_mode: true → simulation_mode: false

# شغل
./scripts/run_all.sh
```

# مثال عملي — يوم كامل بحديقة طماطم 🍅

### الساعة 6 صباحاً ☀️
```
حرارة: 28°C | رطوبة تربة: 55% | pH: 6.5
→ النظام: "كل شي طبيعي ✅"
```

### الساعة 10 صباحاً 🌡️
```
حرارة: 35°C | رطوبة تربة: 40%
→ النظام يشغل المروحة 50% 🌀
→ يتوقع التربة تنشف خلال 45 دقيقة
→ يشغل ري وقائي لمدة 3 دقائق 💧
```

### الساعة 11 صباحاً - 2 ظهراً 🚫
```
→ النظام يوقف الري (الشمس قوية — المياه تتبخر)
→ المروحة تزيد لـ 80%
```

### الساعة 3 عصراً ⚗️
```
pH: 7.2 (المطلوب للطماطم: 6.4)
→ يشغل مضخة pH Down لـ 15 ثانية
→ ينتظر ساعتين قبل الفحص التالي
```

### الساعة 10 مساءاً 🌙
```
→ مروحة تنخفض لـ 30% (وضع ليلي)
→ إضاءة تنطفي
→ ما يشغل رذاذ بالليل (خطر فطريات)
```

**كل هذا تلقائي! 🎉**

# دليل الاستخدام التفصيلي 📖

## شلون تضيف نبات للمزرعة؟

### الخطوة 1: افتح مكتبة النباتات
اضغط زر **"إضافة نبات"** (أو "Add Plant") من الشاشة الرئيسية — موجود في الأعلى يمين أو وسط الصفحة.

```
┌─────────────────────────────────────┐
│  🌿 Smart Garden         [+ إضافة]  │
└─────────────────────────────────────┘
```

### الخطوة 2: ابحث أو اختار النبات
- المكتبة تحتوي **42 نبتة** مقسمة بالفئات:
  - 🌾 حبوب (قمح، أرز، ذرة...)
  - 🥦 خضروات (طماطم، خيار، بطاطا...)
  - 🍓 فواكه (مانجو، فراولة، تفاح...)
  - 🌴 أشجار (تمر، زيتون، نخل...)
  - 🌿 أعشاب (نعناع، ريحان...)
- في خانة البحث اكتب اسم النبتة بالإنجليزي (مثلاً: `tomato`)

### الخطوة 3: اضغط على النبتة
بعد ما تضغط على النبتة اللي تريدها:
- **اختار المنطقة (Zone):** المنطقة 1، 2، الخ (حسب أرضك)
- **اضغط "Add to Farm"** — النبتة تنضاف للشبكة مباشرة!

> ✅ **ملاحظة:** تقدر تضيف أكثر من نبتة بنفس الوقت بمناطق مختلفة.

## شنو يعرضلك بكل بطاقة نبات؟

بعد ما تضيف نبتة، يظهر لك كارت (بطاقة) بالشكل التالي:

```
┌──────────────────────────┐
│ 🍅  Tomato               │
│ Zone 1  ·  Harvest: 90d  │
│                          │
│  ████████░░  78%         │ ← نسبة الصحة (Health Score)
│  ● Thriving              │ ← الحالة الحالية
│                          │
│ [تفاصيل]  [حذف]          │
└──────────────────────────┘
```

| العنصر | شنو يعني |
|--------|---------|
| **نسبة الصحة %** | كيف حال النبتة — 100% ممتاز، أقل من 40% حرج |
| **Thriving ✅** | النبتة بأحسن حال |
| **Stressed ⚠️** | شي ناقص (حرارة / رطوبة / ضوء) |
| **Critical 🔴** | يحتاج تدخل فوري |
| **Dying 💀** | الأوضاع سيئة جداً، راجع الحساسات |
| **Harvest: 90d** | باقي 90 يوم للحصاد |
| **Zone 1** | المنطقة اللي مزروع فيها |

## شلون تشوف تفاصيل النبتة؟

اضغط **"تفاصيل"** أو **"Details"** على أي بطاقة نبات، يفتحلك نافذة تعرض:

1. **مقارنة الظروف الحالية مع المثالية:**
   ```
   الحرارة:     ████████░░  32°C  (المثالي: 25°C) ⚠️
   رطوبة التربة: ██████████  62%   (المثالي: 70%) ✅
   pH:           █████████░  6.8   (المثالي: 6.5) ✅
   الضوء:        ████░░░░░░  18k   (المثالي: 35k) ⚠️
   ```

2. **نصائح العناية** — 3 نصائح خاصة بالنبتة

3. **تاريخ الصحة** — رسم بياني يوضح صحتها خلال الـ 24 ساعة الأخيرة

## شلون تستخدم ESP32 مع المشروع؟

### التوصيل الأساسي (بدون هاردوير كامل)

لو عندك بس ESP32 وحساس رطوبة تربة وحساس حرارة:

**الخطوات:**

#### 1. عدّل ملف الإعدادات
افتح `firmware/config.h` وغيّر:
```c
// ← اسم شبكة الواي فاي مالتك
#define WIFI_SSID     "اسم_الشبكة"
#define WIFI_PASS     "الباسورد"

// ← IP الكمبيوتر اللي شغّلت عليه الباك اند
#define MQTT_SERVER   "192.168.1.100"
#define MQTT_PORT     1883
```

#### 2. شوف IP الكمبيوتر مالتك
```bash
# في الطرفية (Terminal)
ip addr show | grep "inet "
# مثلاً: inet 192.168.1.100
```

#### 3. ارفع الكود للـ ESP32
- وصّله بكيبل USB
- افتح Arduino IDE
- اختار: **Tools → Board → ESP32 Dev Module**
- اختار البورت الصحيح
- اضغط ⬆️ Upload

#### 4. تحقق من الاتصال
افتح Serial Monitor بالـ Arduino IDE (9600 baud)، يجب تشوف:
```
✅ WiFi Connected — IP: 192.168.1.105
✅ MQTT Connected → 192.168.1.100:1883
📡 Publishing sensor data...
```

## كيف يتواصل ESP32 مع الداشبورد؟

```
ESP32
  │
  │  WiFi (كل 5 ثواني)
  ▼
MQTT Broker (Mosquitto)
  │
  ├─→ agrimaster/sensors  ← يستقبل البيانات
  └─→ agrimaster/commands ← يرسل الأوامر
  │
  ▼
FastAPI Backend
  │
  ├─→ SQLite (يخزن البيانات)
  └─→ WebSocket → الداشبورد (يعرض مباشرة)
```

**مثال بيانات تنرسل من ESP32:**
```json
{
  "device_id": "agrimaster_01",
  "sensors": {
    "temperature": 34.5,
    "humidity": 48.0,
    "soil_moisture": 32.0,
    "ph": 6.8,
    "light_lux": 22000
  }
}
```

الداشبورد يحدّث نفسه فوراً لمن يستقبل هالبيانات — **بدون ما تضغط Refresh**.

## تفسير أرقام الحساسات

| الحساس | الرمز | القيمة الطبيعية | متى تقلق؟ |
|--------|-------|----------------|----------|
| 🌡️ الحرارة | °C | 20-35°C | أكثر من 40°C |
| 💧 رطوبة الجو | % | 40-80% | أقل من 30% |
| 🌱 رطوبة التربة | % | 40-70% | أقل من 30% |
| ⚗️ حموضة التربة | pH | 6.0-7.0 | أقل من 5 أو أكثر من 8 |
| ☀️ الضوء | lux | 10,000-60,000 | أقل من 5,000 للنباتات |
| 💨 CO₂ | ppm | 400-1000 | أكثر من 2000 |

## زر تبديل اللغة 🌐

في الداشبورد، ابحث عن زر **EN / AR** في الزاوية اليمنى العليا:
- اضغط **AR** للتحويل للعربية (الصفحة تصير من اليمين لليسار)
- اضغط **EN** للرجوع للإنجليزية
- النظام يتذكر اختيارك حتى بعد ما تقفل المتصفح
